  1245‚Üí  1245‚Üíendpoint rather than querying Prisma directly.

  1247‚Üí  1247‚Üí- **Origin resolution**: uses `process.env.NEXTAUTH_URL` (always set in `.env`

  1250‚Üí  1250‚Üí  behind the Caddy gateway, and `NEXTAUTH_URL` is already the canonical app
  1251‚Üí  1251‚Üí  URL used elsewhere by NextAuth itself.

  1254‚Üí  1254‚Üí  deleted track immediately 404s instead of serving a stale cached share page.

  1256‚Üí  1256‚Üí  `notFound()` (with `console.error`). A broken upstream shouldn't render a

  1258‚Üí  1258‚Üí  in try/catch so a JSON parse failure or connection refused is caught.

  1260‚Üí  1260‚Üí  ‚Äî type-only, so no server code leaks into the page bundle, and the type
  1261‚Üí  1261‚Üí  stays in sync with the API's canonical export. (Same pattern
  1262‚Üí  1262‚Üí  `track-embed.tsx` already uses.)

  1264‚Üí  1264‚Üí#### `generateMetadata` on both pages
  1265‚Üí  1265‚Üí
